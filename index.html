O Bootstrap é um kit de ferramentas de front-end poderoso e repleto de recursos. Construa qualquer coisa — do protótipo à produção — em minutos..

Usado para aplicações rapidas e sem exclusividade

Git, o que é?

Git é um software VCS local que permite aos desenvolvedores salvar snapshots de seus projetos ao longo do tempo. Geralmente é melhor para uso individual

Facilita trabalhos em grupos e a cada edição do codigo salva uma copia

Github

GitHub é uma plataforma baseada na web que incorpora os recursos de controle de versões do git para que possam ser usados colaborativamente.

Uma referência global para repositórios e Network.
